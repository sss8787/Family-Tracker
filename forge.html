Projects

Projects group your characters and their content into separate worlds. The nav selector scopes the roster and all pickers to the active project. Deleting a project never deletes its contents — they move to Unfiled.

Backup & Device Sync

Local backup (one file, everything)

Exports roster, relationships, scenes, arcs, consult threads, and attached stories as a single file. Restore replaces everything on this device with the backup's contents.

GitHub Gist sync (between devices)

Uses a private GitHub Gist as your sync store. Create a token at github.com/settings/tokens with ONLY the "gist" scope, paste it below on each device. Push uploads this device's data; Pull replaces this device's data with the gist's.
